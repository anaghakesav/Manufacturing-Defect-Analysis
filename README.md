# 🏭 Manufacturing Defect Analysis Dashboard

An interactive dashboard built using **Python**, **Streamlit**, **Pandas**, **Matplotlib**, and **Seaborn** to analyze manufacturing defects, monitor production quality, and generate actionable business insights.

---

## 📌 Project Overview

Manufacturing industries generate large volumes of production data. This project focuses on analyzing manufacturing data to identify defect patterns, evaluate machine performance, and support quality improvement through interactive visualizations and analytics.

The dashboard enables users to filter production records, monitor key performance indicators (KPIs), explore defect trends, and gain business insights for better decision-making.

---

## 🚀 Features

### 📊 KPI Monitoring

* Total Products
* Total Defects
* Defect Rate (%)
* Average Temperature

### 🔍 Interactive Filters

* Filter by Machine
* Filter by Shift

### 📈 Visual Analytics

* Product Quality Distribution
* Machine-wise Defect Analysis
* Shift-wise Defect Analysis
* Temperature Distribution
* Correlation Heatmap

### 💡 Business Insights

* Machine with Highest Defects
* Average Temperature
* Average Pressure
* Current Defect Rate

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/Manufacturing-Defect-Analysis.git
```

### Navigate to the Project Folder

```bash
cd Manufacturing-Defect-Analysis
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
